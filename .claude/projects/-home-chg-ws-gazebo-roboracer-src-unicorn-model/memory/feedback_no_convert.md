Do not modify, convert, or compress user-provided files (STL, meshes, images, etc.) without explicit permission.

**Why:** User provided an 8MB STL file and I converted it to binary without asking. The user was upset — 8MB is perfectly fine and the original format matters.

**How to apply:** Always keep original files intact. If optimization seems needed, suggest it to the user first and let them decide.
